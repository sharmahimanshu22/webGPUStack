WebGPUStack

This is right now a template to develop webGPU based webpages.

To run:

1. git clone <repo>
2. cd <root directory of repo>
3. npm install
4. npm run build
5. npm run dev
6. Copy and paste the url returned in previous command to webGPU enabled browser.
